# WP Admin Dark Theme

**WP Admin Dark Theme** is a sleek, highly customizable plugin that completely transforms your default White WordPress admin dashboard into a gorgeous "Eye-Comfortable" Dark Mode layout. It replaces harsh pure-blacks with a carefully calibrated "Blue-Grey" palette designed to reduce eye strain for developers and content creators during long working hours.

## Key Features

* **Global UI Transformation:** Instantly styles the WordPress admin dashboard's background, links, tables, sidebars, top menu, and more.
* **Quick Toggle Switch:** Features a quick-access lightbulb/moon toggle button squarely in the top Admin Bar to fluidly switch back to light mode whenever you need it. 
* **Complete Color Control:** Don't like the default colors? Go to **Settings > Dark Theme** and utilize native Color Pickers to customize every aspect of the theme (Base Background, Borders, Headings, Links, Accent Colors). 
* **TinyMCE Editor Integration:** Not a fan of dark mode while actually writing posts? You can launch the **Editor Display UI** directly from a new button in the TinyMCE toolbar that lets you individually customize your Post Editor's font-size, background color, and text color using a live slider and color pickers. This allows you to write comfortably *without* affecting the HTML that publishes to your live site!

## Installation

1. Upload the `admin-dark-theme` directory to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Access the custom options via **Settings > Dark Theme**, or toggle it on/off via the Admin menu bar.

## Customization

Visit **Settings > Dark Theme** to completely customize your palette. 
Click **"Reset to Defaults"** anytime if you mess up the colors and want to revert to the default Blue-Grey "IDE" layout.

### Editor Working UI Settings
When creating a post, you'll see a new `< / >` Toolbar button in your Visual Editor. Clicking this opens the **Editor Working UI Settings** window, allowing you to fine-tune your working canvas independent of the global dashboard. Use the slider to increase font sizes and the color pickers to tweak the text contrast.
